# Notes Book

A book of notes provided as notebooks (previously built with [Jupyter Book](https://jupyterbook.org/en/stable/intro.html)) built with [Quarto](https://quarto.org/) that you find [here](https://wallytutor.github.io/notes-book/).

## Development

From PowerShell, check the script help with `Get-Help .\develop.ps1 -Full` for details.
